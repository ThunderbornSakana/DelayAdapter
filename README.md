# DelayAdapter
Addressing Delayed Feedback in Conversion Rate Prediction: A Domain Adaptation Approach.

## Requirements
- Python 3.11.7
- PyTorch 2.1.2

## Data Downloading and Preprocessing
First, make the following directories:
- `./data/criteo`
- `./data/taobao`
- `./data/tencent`
- `./checkpoints`

Then, download Criteo from [Criteo](https://labs.criteo.com/2013/12/conversion-logs-dataset/) and put raw data in `./data/criteo`; Similarly, download Taobao and Tencent from [Taobao](https://tianchi.aliyun.com/dataset/649) and [Tencent]([https://tianchi.aliyun.com/dataset/649](https://github.com/guicunbin/Tencent_Social_Advertising_Algorithm_Competition)https://github.com/guicunbin/Tencent_Social_Advertising_Algorithm_Competition), respectively.
